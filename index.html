📥 Fitness Akadémia letöltők

Bookmarkletek az iskola.fitness.hu tananyagok mentéséhez.

📄 Dokumentumok → ZIP

A Dokumentumok oldal összes fájlját egy ZIP-be tölti, a dobozok címe szerinti mappákba rendezve.

  1. Húzd ezt a gombot a könyvjelző sávodra: ⬇ Dokumentumok ZIP
  2. Nyisd meg bejelentkezve a Dokumentumok lapot.
  3. Kattints a könyvjelzőre. A jobb alsó sarokban értesítő mutatja a haladást.
  4. Letöltődik a dokumentumok.zip.

🎬 Online tudástár (videók) → ZIP

Az Online tudástár összes videóját letölti, kurzusonkénti mappákba rendezve.

  1. Húzd ezt a gombot a könyvjelző sávodra: ⬇ Videók ZIP
  2. Nyisd meg bejelentkezve az Online tudástár lapot.
  3. Kattints a könyvjelzőre.
  4. A jobb alsó sarokban látod a haladást (kurzus, aktuális videó, letöltött méret). A lapot hagyd nyitva, amíg fut.
⚠️ 80 videó több GB is lehet — a letöltés eltarthat egy ideig.

Chrome / Edge (vagy bekapcsolt flaggel Brave): egyetlen nagy online-tudastar.zip, egyenesen a lemezre streamelve (nem fogy a memória) — kattintás után rákérdez a mentési helyre.
Brave alapból: a File System Access API tiltva, ezért a fallback kurzusonként külön ZIP-et tölt (pl. online-tudastar - Anatómia_alapszak.zip). Ha az egy nagy ZIP-et szeretnéd: brave://flags → „File System Access API” → Enabled → újraindítás.

Mindkét eszköz azonos eredetből (same-origin), a bejelentkezett munkameneteddel tölt — az élő oldalon kell futtatni, nem a lementett HTML-en.
